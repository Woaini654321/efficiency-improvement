# 销售商机互助平台 — 原型 UI Spec（Vue3 重构参考）

> 从 `PRD/商机平台/living-docs/prototypes/*.html`（React18+antd5 原型）提取的每页 UI 规格。用于 Vue3 重构。登录由框架内置。
> 富文本用 Quill 原型 → Vue3 用 antdv 或封装；echarts 原型多为自绘 SVG/CSS，可换 echarts 或保留。状态原型用 localStorage，Vue3 改 mock/pinia。

## PC-01 找方案（首页/工作台）dashboard
- 区域：Hero搜索(深色渐变) / 公告banner / 快捷操作4卡(查方案·提需求·发布方案·发起讨论) / 我的任务看板 / 方案推荐网格(热门+推荐) / 底部2列(情报中心+讨论热帖)
- Hero：搜索框"搜索方案、需求、讨论话题..."+热门搜索标签(5G模组/Cat.1/车载方案/NB-IoT/智慧城市/天线设计)+4统计(方案总数/待响应需求/本周讨论/活跃用户)
- 任务看板：待处理N/已逾期N；行=优先级点+描述+会议名+截止(超期红闪)+优先级Tag+状态Tag；"查看全部"→PC-22
- 方案推荐：卡片(封面/排名#/订阅徽章/类型Tag/浏览/发布者)；"浏览全部"→PC-17
- 公告banner：类型图标+置顶Tag(红)+标题+类型Tag+发布者+时间+阅读数→PC-16
- 情报区(深色头)3条(品牌徽标+标题+类型+时间)；讨论热帖(话题Tag+标题+作者/回复/浏览/日期)
- badges：方案类型 产品信息=blue/解决方案=green/成功案例=orange；任务优先级 特急=红/紧急=橙/普通=灰；任务状态 待处理=blue/处理中=orange/已完成=green/已转交=purple；公告类型 通知=blue/政策=purple/活动=green/其他=default

## PC-02 商机详情页（方案详情）detail
- 区域：过期提示横幅/面包屑+返回/Hero卡(类型Tag+状态Tag+标题+发布者+分类Tag+指标chips+操作)/正文富文本/附件区/互动栏/评论区/相关方案推荐/分享弹窗

- 互动栏：点赞/收藏/关注/分享
- 附件行：文件图标(pdf红/pptx橙)+名+大小+下载+预览
- 评论区：公众号风格2级扁平(顶级评论卡:头像/部门Tag/时间/内容/点赞/回复 + 扁平回复列表)；TextArea带@提及下拉；"展开更多评论"

## PC-03 创建/编辑商机（发布方案表单）form
- 区域：sticky标题栏(返回+标题+自动保存指示)/copy_from Alert/基础信息卡/方案详情富文本卡/附件卡/固定底部操作栏

- 附件：点击上传(PDF/DOC/XLS/PPT/JPG/PNG/ZIP,单≤50MB总≤200MB)
- 操作：保存草稿/发布方案(代为发布/发布为新方案)

## PC-05 需求详情页 detail

## PC-06 发布/编辑需求 form
- 区域：标题栏/角色权限Alert/基础信息卡/可见范围卡/邀请产品线卡/需求详情富文本卡/附件卡/相似需求检测区/固定底部操作栏
- 基础信息：标题(Input≤100 必填)/紧急程度(Select 普通/紧急/特急 必填,特急红警告)/期望响应截止时间(DatePicker showTime)/行业场景(Input≤50)/关键词(Select tags)/分类标签(Cascader multiple 必填 1~5)

- 邀请产品线：Cascader multiple(留空自动匹配)
- 需求详情：Quill(必填)；附件：Dragger(≤50MB≤10个)
- 相似需求检测：标题防抖600ms检测,显示匹配(相似%+状态Tag+查看)，无匹配绿OK
- 操作：发布/保存(Modal确认含摘要+相似警告+同步邮件)，编辑?mode=edit回填，离开确认
